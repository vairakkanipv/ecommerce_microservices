Step 1 — Get the token and save it to a variable
bashcurl -X POST http://localhost:8001/api/token/ -H "Content-Type: application/json" -d "{\"username\": \"admin\", \"password\": \"admin123\"}"
You'll see the full token response. Now copy just the access token value (the long string after "access":").

Step 2 — Store it in a variable in your terminal
In Cmder / PowerShell:
bashset TOKEN=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ0b2tlbl90eXBlIjoiYWNjZXNzIiwiZXhwIjoxNzc0OTMxNDM3LCJpYXQiOjE3NzQ5Mjc4MzcsImp0aSI6ImNmZjMyZWE2OGM0NDRmZWM5ODgyYzAwNjdkMDMzYTEzIiwidXNlcl9pZCI6MX0.YOUR_FULL_TOKEN_HERE
Paste your complete token — the whole thing with no line breaks.

Step 3 — Use the variable in your curl commands
bashcurl http://localhost:8001/api/products/ -H "Authorization: Bearer %TOKEN%"
bashcurl http://localhost:8001/api/products/1/check_stock/ -H "Authorization: Bearer %TOKEN%"
bashcurl -X POST http://localhost:8002/api/orders/ -H "Content-Type: application/json" -H "Authorization: Bearer %TOKEN%" -d "{\"product_id\": 1, \"quantity\": 2}"
